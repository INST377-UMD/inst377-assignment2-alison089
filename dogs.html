Dog Breeds

In this web page there are options to Navigate using Audio!

Ensure your Mic is enabled to be able to use this feature.

  • Navigate to a Different Page
    • Say "Navigate to <page>"
  • Change Your Page Color
    • Say "Change the color to <color>"
  • Say Hello
    • Say "Hello"

Voice Commands

You can use the following voice commands on this website:

  • "Hello" - Will display a Hello World alert
  • "Change the color to [color]" - Changes the background color
  • "Navigate to [page]" - Navigate to Home, Stocks, or Dogs page
